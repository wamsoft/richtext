# ビルド方法

CMake で構造が構築されています。Makefile で基本的なビルド用の呼び出しが定義されています

```
# CMAKE処理呼び出し
make prebuild   

ビルド実行
make build
```

サブディレクトリの ext/minikin と ext/thorovg はビルドが通る状態になっています

CMakeLists.txt の  krkrz_plugin(richtext ...) の部分に作成したソースを追加していって、
make prebuild && make build でビルド確認可能です

- `<s>` / `<strike>` / `<del>` 取り消し線

- `<sup>` 上付き文字
- `<sub>` 下付き文字
- `<style name='...'>` 名前付きスタイル参照
- `<br>` 改行
- `<sp width=...>` スペース
- `<outline color=... width=...>` 縁取り
- `<shadow color=... x=... y=...>` 影

エスケープシーケンス: `&lt;`, `&gt;`, `&amp;`, `&quot;`, `&apos;`, `&nbsp;`, `&#nnnn;`, `&#xhhhh;`

ビルド確認: 成功

# ビルド構造

```
richtext_lib (静的ライブラリ)
    ├── FontFace.cpp
    ├── FontManager.cpp
    ├── TextStyle.cpp
    ├── Appearance.cpp
    ├── TextLayout.cpp
    ├── ParagraphLayout.cpp
    ├── GlyphRenderer.cpp
    ├── TextRenderer.cpp
    └── TagParser.cpp
    
richtext.dll (吉里吉里プラグイン)
    ├── main.cpp (TJS バインディング)
    └── → richtext_lib をリンク

sample_render.exe (サンプル実行ファイル)
    ├── sample/sample_render.cpp
    └── → richtext_lib をリンク
```
