# 実装進捗

## ビルド方法

```bash
# 事前準備: VCPKG_ROOT 環境変数を設定
# テスト用フォントのダウンロード
make fontdata

# CMake 構成（初回または CMakeLists.txt 変更時）
make prebuild                      # OS 自動検出
make prebuild PRESET=x64-windows   # プリセット明示指定

# ビルド
make build
make build BUILD_TYPE=Debug        # デバッグビルド

# サンプル実行（プロジェクトルートから）
./build/x64-windows/Release/sample_render.exe
# → output.bmp が生成される
```

## 既知の課題

### 太字・斜体のフェイク描画が未実装

minikin は `FontFakery` で太字（bold）・斜体（italic）のフェイクフラグを計算するが、`GlyphRenderer` がこのフラグを参照していない。そのため `<b>` / `<i>` タグは TextStyle のウェイト/イタリックを設定するだけで、実際の描画に反映されない。

対応方針:
- **太字**: `GlyphRenderer` で `fakery.isFakeBold()` が true の場合、FreeType の `FT_Outline_Embolden` でアウトラインを太らせるか、Stroke を追加して太字効果を模擬
- **斜体**: `fakery.isFakeItalic()` が true の場合、FreeType の `FT_Matrix` で変換行列を適用してシアー（傾き）を加える
- 別途 Bold/Italic フォントファイルを登録すれば、minikin のフォント選択で対応可能（フェイク不要）
